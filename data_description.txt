Данные представляют собой все турниры по интеллектуальным играм, которые были загружены на сайт rating.chgk.info в период с 1 сентября 2012 по 29 ноября 2021 года.

Данные разбиты на три файла. Файл tournaments.csv содержит описание турниров и содержит 6 полей.

id -- универсальный идентификатор турнира
name -- название турнира
dateStart -- дата начала турнира
dateEnd -- дата окончания турнира
type -- тип турнира и его кодовое обозначение
questionQty -- количество вопросов в каждом туре

Файл results.csv содержит результаты всех турниров. Для каждой команды участвовавшей в турнире в файле содержится запись со следующими полями:
tournament_id -- универсальный идентификатор турнира
team_id -- идентификатор команды, может повторяться от турнира к турниру, что позволяет оценивать командные результаты и их динамику
name -- название команды
town -- город, который представляет команда
mask -- повопросные результаты команды в турнире, для каждого вопроса может принимать следуюшие значения: '1' -- команда взяла вопрос, '0' -- команда не взяла вопрос, '-' -- вопрос не взяла ни одна команда на турнире   questionsTotal -- количество правильных ответов на вопросы в рамках турнира
position -- занятое командой место

Файл roster.csv содержит составы комадн на каждом из турниров. Для каждого игрока, участвовавшего в турнире, указывается команда за которую он играл. Данный файл содержит следующие поля:
tournament_id -- идентификатор турнира
team_id -- идентификатор команды
player_id -- универсальный идентификатор игрока. Каждый игрок соответствует одному уникальному идентификатору, что позволяет отслеживать личные результаты
flag -- в этом поле может быть указана роль игрока в команде. 'К' -- капитан команды, 'Б' -- игрок входящий в базовый состав команды, 'Л' -- легионер, игрок временно играющий за данную команду
